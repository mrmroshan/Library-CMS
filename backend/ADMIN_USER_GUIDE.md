# Admin User Guide

This guide explains how to operate the University Community Library CMS as an administrator.

## 1. Login

1. Open `/login`.
2. Enter your admin email and password.
3. After login, open **Admin** from top navigation.

## 2. Dashboard Overview

The dashboard shows:
- Total users and active users
- Total articles and pinned articles
- Total library files and download count

Use this page as a quick health check.

## 3. Manage Categories

Open **Manage PDF Library** and create categories (for example: Medical, Science, Engineering, General).

Rules:
- Every uploaded PDF should have a category.
- A category can be deleted only when no books are assigned to it.

## 4. Upload and Maintain PDFs

In **Manage PDF Library**:
1. Select category.
2. Upload PDF.
3. Add title, author, and description.
4. Keep visibility ON to show in public library listing.

Actions:
- **Hide**: temporarily removes a book from public list.
- **Remove**: permanently deletes the book.

## 5. Manage Articles

In **Manage Articles**:
- Create/update/delete articles.
- Use **Pin on top** for important announcements.
- Keep only high-priority posts pinned.

## 6. Manage Users

In **Manage Users**:
- Update user name/email.
- Set role to `admin` or `student`.
- Activate/deactivate accounts.

Best practice:
- Keep admin accounts limited to trusted team members.

## 7. Homepage QA Checklist

After updates, verify:
- Category filter chips are working.
- Library cards display correctly.
- Non-logged users are prompted to login for downloads.
- Logged-in users can download visible PDFs.

## 8. Operating Tips

- Use clear and consistent category names.
- Remove duplicate books.
- Keep descriptions concise.
- Regularly review inactive or suspicious user accounts.
